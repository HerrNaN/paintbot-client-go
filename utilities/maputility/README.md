# Map utility
A utility package for using the paint bot map objects in a more programmer friendly format. 
For example it provides conversion functions from positions to coordinates.

### Positions
``` 
0 1 2
3 4 5
6 7 8
```

### Coordinates {X, Y}
```
{0,0} {1, 0}, {2, 0}
{0,1} {1, 1}, {2, 1}
{0,2} {1, 2}, {2, 2}
